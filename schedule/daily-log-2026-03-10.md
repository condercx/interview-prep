### 💡 知识点记录 / 重点摘录
- `const` 使用场景：可修饰一般常量、数组、（C++）对象/引用、指针、函数形参与返回值；其中返回普通值类型时加 `const` 通常意义不大。
- `new/delete` 与 `malloc/free` 核心区别：前者是 C++ 运算符、具备类型安全并触发构造/析构；后者是 C 函数，仅做内存分配释放，不调用构造/析构，且两组 API 不能混用。
- `strlen("\0")` 与 `sizeof("\0")`：`strlen` 统计终止符前字符数，所以为 `0`；`sizeof` 统计字面量数组总大小，`"\0"` 含两个空字符（转义字符 + 结尾补零），所以为 `2`。
- `CPU/MPU/MCU/SoC/SoPC` 关系：从“计算核心”到“系统集成”再到“可编程硬件”逐步演进；面试高频对比点是是否内置 RAM/ROM、是否可最小系统直接运行、系统复杂度、硬件是否可重构。
- 记忆句：`CPU` 是核，`MPU` 是强核但常需外扩，`MCU` 是单片小系统，`SoC` 是单片大系统，`SoPC` 是可重构系统芯片。
