#### 刷题
- [x] 代码随想录 [1]：[两数之和]
  **思路：** 用 unordered_map 存储遍历过的元素 每次查找 target - nums[i] 是否存在 找到则返回两个下标
	- **为什么用 unordered_map：** 需要同时记录： ├── 元素的值 → 作为 key └── 元素的下标 → 作为 value 数组 → 大小受限，浪费内存 set → 只能存key，无法记录下标 map → key存值，value存下标 ✅
	- **流程：** nums = [2, 7, 11, 15] target = 9 i=0: 查找 9-2=7 map={} 没找到 → 存入{2:0} i=1: 查找 9-7=2 map={2:0} 找到！ → 返回{0,1}
	- **时间复杂度：** O(n)：只需遍历一次 空间 O(n)：map最多存n个元素

  **代码位置：**`code/leetcode/2026-05-07-two-sum.cpp`
