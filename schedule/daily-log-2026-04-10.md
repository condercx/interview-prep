### ❓ 遇到的问题 / 未解决的疑问
❌ 错误写法
```c
int *ans = (int*)malloc(sizeof(int) * 2);
// ans[0] 和 ans[1] 是随机脏数据！
// 导致输出: [-1094795589,-1094795588]
```
✅ 正确写法
```c
int *ans = (int*)calloc(2, sizeof(int));
// ans[0] 和 ans[1] 自动初始化为0
// 异或运算结果正确！
```
