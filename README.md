# 📚 嵌入式软件工程师秋招复习仓库

本仓库用于系统准备 **2026 届秋季校园招聘**，涵盖嵌入式软件开发岗位所需的核心知识体系。  
复习计划周期为 16 周，结合《嵌入式面试宝典》与 LeetCode 刷题，目标做到理解透彻、手写熟练、表达清晰。

---

## 📅 总体复习阶段

| 阶段 | 周次 | 核心任务 |
|------|------|----------|
| 第一阶段 | 1~4 周 | C/C++ 基础 + 数据结构（链表、树、排序） |
| 第二阶段 | 5~8 周 | 操作系统核心 + ARM 体系 + Linux 基础 |
| 第三阶段 | 9~12 周 | 网络编程 + Linux 驱动入门 + 项目实践 |
| 第四阶段 | 13~16 周 | 高频算法二刷 + 模拟面试 + 查漏补缺 |

详细周计划请见 [`schedule/weekly-plan.md`](schedule/weekly-plan.md)

## 🚀 使用说明

1. **每日学习**：按照周计划推进，学习笔记放入 `notes/` 对应目录，代码练习放入 `code/`。
2. **每日日志**：在 `schedule/daily-log.md` 中记录当天内容、问题及明日计划。
3. **周末复盘**：回顾本周笔记，整理错题，更新周计划完成情况。
4. **同步远程**：每天结束时执行 `git add .` / `git commit` / `git push` 备份进度。

## 📌 注意事项

- **主线优先**：每天保证 1 小时 C/C++ 主线学习，支线内容穿插进行。
- **理解为主**：面试八股文要在理解基础上记忆，切忌死记硬背。
- **坚持刷题**：每天至少 1 道 LeetCode，保持手感。
- **项目复盘**：每周抽时间用 STAR 法则梳理一个项目细节。
